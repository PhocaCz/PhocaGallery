# Phoca Gallery

## Phoca Gallery — Community-Driven Extension for Joomla CMS

Phoca Gallery is an efficient image gallery extension for Joomla! CMS.

### Free and Open Source Software (FOSS) Declaration

This project is developed, distributed, and maintained strictly as **Free and Open Source Software (FOSS)**. It is provided entirely outside the course of any commercial activity as an independent public resource.

Our compliance with the principles of free and open-source software is established by the following structural and operational guidelines:

* **Strictly Non-Commercial & Public Good:** This software is provided 100% free of charge. No commercial entities control or fund this repository, and there are no commercial monetization models, paid subscriptions, or premium paywalls associated with this codebase.
* **Independent Volunteer Contributions:** The development, maintenance, and testing of this extension are driven entirely by independent, voluntary contributions from the global community.
* **No Commercial Technical Support:** Technical support is not provided as a paid commercial service by the maintainers of this repository. Community assistance is available on a voluntary, non-binding basis via public forums.
* **No Proprietary Lock-in:** The software is licensed under the copyleft GNU General Public License (GPL). You have absolute freedom to run, study, share, modify, and redistribute the code.
* **Decentralized Infrastructure:** This codebase is hosted in a public, transparent repository on GitHub to foster open collaborative development and public peer review, ensuring safety and quality through community vigilance.

*This repository is a shared community resource dedicated to the global open-source ecosystem.*

### Community Feedback & Contributions
